# 自定义词典 / 热词
#
# 每次润色时（LLM 之前依次执行）：
#   ① 软术语：作为 <词表> 喂给 LLM，遇到同音/近音词时它会优先采用、并保留术语原样；
#   ② 硬替换：确定性整串替换（只放几乎不会误伤的固定错）；
#   ③ 拼音纠错词：把「读音相同、字面不同」的连续汉字片段强制纠成正确词
#      （一条顶所有同音变体，且对所有风格含纯逐字稿都生效）。
#
# 语法（每行）：
#   词            —— 软术语：术语的正确写法（结合上下文 + 拼音由 LLM 判断）
#   误识 => 正确   —— 硬替换：确定性把"误识"换成"正确"
#   ~正确词        —— 拼音纠错词：凡读音同它的连续汉字片段都纠成它（需 ≥2 字；自身也进软词表）
#   # 开头为注释；空行忽略
#
# 提示：像"流逝""油画"这种本身是常用词的，别放硬替换也别加 ~（会误伤"时光流逝""一幅油画"），
#       放软术语让 LLM 结合上下文判断即可。~ 只给"常被同音误识、自身又不是日常常用词"的词。
